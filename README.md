# Ghost Closet

Dark, interactive storefront and campaign experience for Ghost Closet.

Live store: <https://ghostxlebanon.github.io/GhostCloset---GC/>

Animated story: <https://ghostxlebanon.github.io/GhostCloset---GC/story/>

## Ownership and access

This project is owner-controlled by `ghostxlebanon`. Repository changes are authorized only from the owner account and GitHub Apps explicitly connected by the owner. The deployment workflow has read-only repository access and cannot commit or push source changes.

## Security model

- Static GitHub Pages deployment over HTTPS.
- No card numbers, expiry dates, CVC codes, passwords, or checkout addresses are collected by Ghost Closet.
- Available products route to official seller websites for payment and delivery.
- Build assets are committed and reviewed in this repository; deployment does not retrieve them from an external build host.
- The GitHub Actions deployment token has read-only access to repository contents.

See [SECURITY.md](SECURITY.md) for vulnerability reporting.
